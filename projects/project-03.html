Project 03

Experimental Interface

An exploration of unusual input methods, sensor behaviour, and interaction patterns beyond standard controls.

Problem / Objective

Test interactions that feel surprising without feeling confusing.

The study looks at how sensors, gesture-like inputs, and feedback loops can create distinct behaviour while staying legible to the user.

Process

Prototype fast, observe responses, keep the signal.

The workflow uses quick test rigs, visual state mapping, and simple interaction choreography to separate novelty from useful behaviour.

Technical Notes

Category: Interface research

Focus: Sensor behaviour, input legibility, response timing

Methods: Test rigs, observation, interaction tuning

Outcome / Next Steps

The structure is ready for motion references, circuit diagrams, and annotated interaction maps. The next step would be narrowing the strongest behaviour into a more product-like prototype.
