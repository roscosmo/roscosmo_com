Workflow

Use this with Pinegrow or DevTools if you want even more control.

The key difference now is that the taste lives in shared tokens, not scattered across the HTML. Small iterations should stay fast.

1. Choose a preset direction.

2. Adjust color, radius, width, and typography.

3. Open other pages in another tab to check the mood in context.

4. Copy the exported CSS block and send it back when a direction lands.

Theme Controls

One place for the visual decisions.

Preset changes reset manual overrides. Manual changes are saved in this browser so you can compare pages without redoing the controls.

These changes do not write back to the repository by themselves. They are meant for rapid visual iteration, then promotion into the theme files once the direction is right.
